# Features Overview

Aerion packs essential email features into a lightweight, efficient package. Here's what you can do.

## Email Management

- **Unified Inbox** - View emails from all accounts in one place, color-coded by account
- **Conversation Threading** - Related messages grouped together for context
- **Folder Navigation** - Full access to all your email folders
- **Search** - Find emails quickly with basic full-text search
- **Bulk Actions** - Select multiple messages and apply actions at once

[Learn more about Email Management →](/docs/features/email-management)

## Composing

- **Rich Text Editor** - Bold, italic, headings, lists, links, tables, and more
- **Plain Text Mode** - Toggle to plain text when needed
- **Attachments** - Click to attach, drag and drop, or paste images
- **Detachable Composer** - Pop out to a separate window
- **Signatures** - Per-identity signature support
- **Draft Auto-Save** - Never lose your work

[Learn more about Composing →](/docs/features/composing)

## Contacts

- **CardDAV Sync** - Connect to Nextcloud, Radicale, or any CardDAV server
- **Google Contacts** - Sync with your Gmail account
- **Microsoft Contacts** - Sync with your Outlook account
- **Autocomplete** - Contacts appear as you type recipient addresses

[Learn more about Contacts →](/docs/features/contacts)

## Keyboard Navigation

Aerion is fully keyboard-navigable with vim-style shortcuts:

- Navigate between panes with `Alt+H/L`
- Move through messages with `J/K`
- Quick actions: `S` to star, `Delete` to trash, `Ctrl+K` to archive
- Compose with `Ctrl+N`, send with `Ctrl+Enter`

[See all Keyboard Shortcuts →](/docs/features/keyboard-shortcuts)

## Privacy Features

- **Local Storage** - All data stays on your computer
- **Direct Connections** - Connects directly to your email provider via IMAP/SMTP
- **Tracking Removal** - Basic removal of tracking elements in emails
- **No Analytics** - Aerion doesn't phone home

## Customization

- **Themes** - Light, Dark, or follow your system preference
- **Message Density** - Choose from Micro, Compact, Standard, or Large
- **Account Colors** - Color-code your accounts for easy identification
- **Per-Account Identities** - Different signatures and sender names per account

[Learn more about Settings →](/docs/features/settings)

## Platform Features

### Linux
- Clickable notifications that focus the relevant email
- Auto-sync when system wakes from suspend
- AppImage for easy installation

### macOS & Windows
- Native application feel
- System theme integration
- Standard installation methods

## What's Not Included

Aerion focuses on core email functionality. These features are not currently available:

- Calendar integration
- PGP encryption
- Email filters/rules
- Offline sending (no message queue - requires connection to send)
- Import from other email clients

Some of these are on the [roadmap](https://github.com/hkdb/aerion) for future releases.
